Introduction

Base URL: http://localhost:8000

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Currency Management

APIs for managing currencies and exchange rates

List all currencies

Returns a list of all available currencies in the system.

Example request:
curl --request GET \
    --get "http://localhost:8000/api/v1/currencies" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost:8000/api/v1/currencies"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Example response (200):


{
    "data": [
        {
            "id": 1,
            "code": "USD",
            "name": "US Dollar",
            "symbol": "$",
            "exchange_rate": 1,
            "surcharge_rate": 0.05
        }
    ]
}

Get Foreign Currency Amount

Convert a USD amount to a foreign currency using current exchange rates.

Example request:
curl --request POST \
    "http://localhost:8000/api/v1/currencies/get-foreign-currency-amount" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"total_amount\": \"100.50\",
    \"currency\": \"EUR\"
}"

const url = new URL(
    "http://localhost:8000/api/v1/currencies/get-foreign-currency-amount"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "total_amount": "100.50",
    "currency": "EUR"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Example response (200):


{
    "data": {
        "foreign_currency_amount": "85.42",
        "total_amount": "100.50",
        "exchange_rate": "0.85",
        "surcharge_rate": "0.05",
        "currency": "EUR"
    }
}

Example response (404, Currency not found):


{
    "error": "Currency not found"
}

Example response (422, Validation Error):


{
    "message": "The given data was invalid.",
    "errors": {
        "currency": [
            "The currency field is required."
        ],
        "total_amount": [
            "The total amount field is required."
        ]
    }
}
Received response:
Request failed with error:
